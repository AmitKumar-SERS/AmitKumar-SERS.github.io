## Running the Website Locally

To preview the website locally before publishing changes, clone the repository and run:

```bash
bundle install
bundle exec jekyll serve
```

Then open the local site in your browser at:

```bash
http://localhost:4000
```

If there are dependency issues, delete `Gemfile.lock` and run `bundle install` again.
